```
NAME
    murder - Sweet gem to create a great party game.

USAGE
    murder <subcommand> [options]

DESCRIPTION
    murder is a suite of commands to help deploy and manage plots and
    character profiles for murder mystery dinners.

COMMANDS
    generate     Generate a new game configuration
    graph        Create murder graphs
    player       Rename a player

OPTIONS
    -h --help     Show help for this command
    -t --trace    Display stack traces on application crash
```
